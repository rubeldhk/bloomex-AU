Contacts XTD - Extended Contacts

Author: Kurt Banfi
Email: mambo@clockbit.com
Website: www.clockbit.com

Contacts XTD is an extended contacts component for Mambo 4.5.x and derived from the original Mambo Contacts Component.

Version: 1.0.1

The following additional information has been included in this release:
  • company name
  • website info
  • second telephone number
  • first & second mobile phone number
  • second miscellaneous info

Installation of the Contacts XTD Searchbot:

Mambos integrated searchbots do not search the extended contacts of this component.
Please visit http://www.clockbit.com and get the Searchbot for this release, then install it via Mambots - Install Mambots.

Don't forget to activate the new Bot after installation otherwise it won't work!

Installation of the Main Menu Extensions:

In order to use this component in its intended way, you will have to download and install the appropriate Contacts XTD Menu Extensions for this release:
  • Download the extensions from http://www.clockbit.com

  • To install them, simply unzip the downloaded file in your Mambo root directory. This will create several new subdirectories below administrator/components/com_menus.
    As there is no official way on how to install such extensions, it has to be done manually. Maybe this feature will be available in future versions of Mambo.

  • If you go to Menu - YOUR_MENU - New, you will notice that it is now possible to create menu entries of type

    • "Link - Contacts XTD Item"

    • "Table - Contacts XTD Category" and

    • "Blog - Contacts XTD Category".
