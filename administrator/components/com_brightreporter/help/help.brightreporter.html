Brightcode Reporter Help

Even tables from other components like community builder.

Query samples:

All users with all fields
Select * from #__users

Users last visit
SELECT `name`, `lastvisitdate` from #__users ORDER BY `lastvisitdate`

Show double users in Community builder (using first- lastname)
select firstname, lastname ,count(*) as no from #__comprofiler group by firstname, lastname having n > 1

#_ is your table extention (e.g. jos_users), you can leave it like this.

A query with params looks like this:

SELECT `name`, `lastvisitdate` from #__users WHERE DATE(lastvisitdate) = '${FirstDate}' OR DATE(lastvisitdate) = '${SecondDate}' ORDER BY `lastvisitdate`

Where ${FirstDate} and ${SecondDate} are the params.

Do you want to use a Select box as a param from a query ?

Please create it like the following query

Select car_id as id, name as value from #__users

Where value is the id to search for and value to show a normal readable value to choose from in the select box.
